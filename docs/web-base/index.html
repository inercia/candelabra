Versatile and Configurable

Candelabra lets you build from simple one-machine topologies to complex layouts with multiple servers. Candelabra can be used for defining simple development topologies (for example, for testing your software in your own machine) and then deploy it to production environments with just one command

Designed for the cloud

Caldelabra's topologies can be used both for your local VirtualBox deployments as for real deployments in the cloud.

Extensible

Candelabra can be easily extended with plugins. Or you can distribute Candelabra topologies with your own hooks for running at some point.
